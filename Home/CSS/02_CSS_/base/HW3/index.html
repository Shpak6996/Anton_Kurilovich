Блок навигации

  • Ширна 15 em
  • Должен присуствовать цвет фона
  • Распологатся слева
  • Блок навигации наполните контентов достаточным, чтобы по высоте он начинался захватывать творую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех "статьях" сделайте отступы снизу от соседней равными .2em, при помощи свойства margin-boottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех "статьях" сделайте отступы снизу от соседней равными .2em, при помощи свойства margin-
boottom. Картинка вставлена при помощи тега, плавание организовано по
правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутри контентом.
